# hpc-learning

## Overview
This repository is my personal learning space for high-performance computing (HPC).  
It contains small projects, experiments, and practice code focused on numerical computing and performance-oriented C programming.

## Goals
- Build a solid foundation in HPC concepts
- Implement numerical algorithms in C
- Learn performance measurement and optimization techniques
- Explore parallel programming (e.g., OpenMP, MPI)

## Repository Structure
- `Numerical-Practice/`
  - `matrix/` — A C matrix library with tests

## Build / Run
Each subproject has its own build instructions.  
Please check the README inside the corresponding directory.
